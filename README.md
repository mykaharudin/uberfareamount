## Instalasi

1. Clone repository ke komputer lokal anda menggunakan perintah berikut:

   ```shell
   git clone https://github.com/mykaharudin/
   ```
   
2. Lakukan Instalasi Library Dengan Perintah Berikut:

    ># shell
    pip install 
    numpy
    pandas
    matplotlib
    scipy
    streamlit
    seaborn
    

## Penggunaan
1. Melakukan Akses dan Kompilasi(Local):
